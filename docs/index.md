# 🧪 Networking Lab – Manual de Implementación Profesional

Bienvenido al manual práctico donde **construirás una infraestructura de red empresarial completa** paso a paso.

## Metodología

Este proyecto consta de **15 laboratorios** que deben realizarse en orden secuencial. Cada laboratorio:

1.  Estudia un tema teórico.
2.  Identifica los conceptos implementables.
3.  Diseña un escenario empresarial realista.
4.  Diseña la topología y el direccionamiento.
5.  Implementa la configuración paso a paso.
6.  Explica cada procedimiento.
7.  Verifica cada etapa.
8.  Muestra el resultado esperado.
9.  Prepara el siguiente laboratorio utilizando el estado final del anterior.

Al finalizar el laboratorio 15, **tendrás una red funcional completa** que integra routing, switching, VLANs, VPN, seguridad, monitoreo y más.

## Requisitos

-   Un sistema Linux (puede ser una VM, WSL o una instalación real).
-   Conocimientos básicos de línea de comandos.
-   Ganas de aprender haciendo.

## Navegación

Usa el menú lateral para acceder a cada laboratorio. **Comienza por el Laboratorio 01** y no saltes ninguno.

¡Manos a la obra!
